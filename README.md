# demo-redux-toolkit-redux-saga
@reduxjs/toolkit + redux-saga

## Dependencies
```bash
yarn add react-redux redux-saga saga-toolkit &&
yarn add -D @reduxjs/toolkit 
```
## Restore package
```bash
npm install
```
## Run on development mode
```bash
npm run dev
```
